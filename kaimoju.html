カイモジュ

KAIMOJU ✦ kaomoji + kaiju
(顔文字 + 怪獣)

A KAOMOJI is a Japanese-style emoticon built from text.
Yours has gone full KAIJU! Smash the city by typing the romaji on each katakana / hiragana tile.

SELECT DIFFICULTY

HOW TO PLAY

GOAL — Smash every building by typing the romaji for the highlighted kana cell.

HP — You start with 5 hearts. A wrong romaji costs one heart. Run out and the game ends.

HINT DELAY — On each new target the romaji is hidden for ~1 second; try to recall it on your own. New characters introduced each level get a one-shot hint reveal.

LEVELS — Kana modes start with 6 characters and add one more per level (gojuon order: a, i, u, e, o, ka, ki, ku…). Dakuon / handakuon come last.

FIND — Chain 10 clean hits in a row to unlock the laser burst. Tap the FIND button at the bottom of the panel to activate, then type the romaji once to detonate every copy on screen. One miss resets the streak. Locked while a brand-new character is your target so you have to learn it first.

COMBO — Consecutive hits build a score multiplier. Any miss drops it back to zero.

☀ / ☾ — Toggle day / night theme.

CTRL+M (or system mute) — Toggle sound.
