What is Spin Nids?

Spin Nids is an ambigram word game. Each tile on the grid holds four letters — one in each corner. Rotate a tile and its letters transform into their ambigram equivalents: n → u, m → w, d → p, and more.

How to play

  • Type letters to build a word from the grid
  • Press Enter or the ✓ button to submit a word
  • Press ⌫ Clear to clear your current word
  • Use ⟷ Flip to mirror the tile hand
  • Drag tiles from the hand onto the grid

Modes

  • Any Letter — use any letter anywhere (beginner)
  • One Per Tile — max one letter per tile per word
  • Adjacent — each letter must touch the previous one
  • Destroy — used letters are destroyed; clear a tile to spawn a new one
